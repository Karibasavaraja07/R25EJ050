System Role / Objective: Act as an expert C programmer. Your task is to write a complete, interactive, text-based 2D graphics engine in C that renders shapes onto a console-based canvas.

Core Architecture & Data Structures:

The Canvas: Create a global 2D character array named canvas with dimensions 25 rows by 60 columns.
Initialization: Provide a function to initialize the entire canvas with empty space characters (' ').
Display: Create a function to render the canvas to the terminal.
Dynamically clear the console screen before rendering (handling both Windows and Unix environments).

Drawing Functions (Requirements & Constraints): All drawing functions must include strict boundary/safety checks to ensure that drawing outside the 25x60 canvas does not cause memory corruption or segmentation faults.

Clear Entire Screen: Reinitializes the entire canvas.
Exit: Breaks the loop and ends the program.
