Here is an example of what the console output looks like when you compile and run the C program.

Because the program is interactive and runs in a loop, here is a step-by-step simulation of the output from starting the program, drawing a rectangle, and seeing the result.

1. Initial Screen
When you first run the program, it displays the menu and an empty 25x60 canvas enclosed in borders:
